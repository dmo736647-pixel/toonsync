# 项目分析报告：AI Webtoon Video Maker

## 1. 项目概览

这是一个名为 **AI Webtoon Video Maker** 的全栈 AI 应用平台，旨在帮助创作者生成具有**角色一致性**和**多语言口型同步**功能的 Webtoon/Manga 视频。

* **核心价值**：解决了 AI 视频生成中常见的角色形象不统一和口型不匹配的问题。

* **目标用户**：独立创作者、小型工作室及企业营销团队。

## 2. 技术架构

项目采用典型的前后端分离架构，并在基础设施层面支持容器化和云原生部署。

### 🏗️ 后端 (Python/FastAPI)

* **核心框架**：[FastAPI](file:///d%3A/ceshi/manju/app/main.py) (高性能异步框架)

* **数据层**：

  * **数据库**：PostgreSQL (通过 Supabase 托管)，使用 [SQLAlchemy](file:///d%3A/ceshi/manju/requirements.txt) ORM 和 Alembic 进行迁移管理。

  * **缓存**：Redis，用于会话管理和任务队列。

* **AI 能力集成**：

  * **Replicate**：托管主流 AI 模型（如 Stable Diffusion 用于图像生成）。

  * **ElevenLabs & Azure TTS**：提供高质量的多语言语音合成。

  * **Wav2Lip**：实现通用的口型同步。

  * **OpenAI**：可能用于剧本生成或辅助功能。

* **异步任务**：使用 Celery 处理耗时的 AI 生成任务（视频渲染、口型同步）。

### 🖥️ 前端 (React/TypeScript)

* **核心栈**：React 18 + TypeScript + Vite。

* **UI/UX**：Tailwind CSS + Headless UI，提供现代化且响应式的界面。

* **状态与交互**：使用 Axios 进行 API 通信，包含 WebSocket 支持（用于实时反馈生成进度）。

### 🚀 基础设施与运维

* **容器化**：提供 `Dockerfile` 和 `docker-compose.yml`，支持本地一键启动。

* **Kubernetes**：`k8s/` 目录包含完整的 K8s 部署配置（Deployment, Service, Ingress）。

* **CI/CD**：GitHub Actions (`.github/workflows`) 实现自动测试和部署。

* **监控**：集成 Prometheus 和 Grafana (见 `monitoring/` 相关文件及中间件配置)。

## 3. 核心功能模块

根据后端路由结构 ([app/main.py](file:///d%3A/ceshi/manju/app/main.py))，主要模块包括：

* **Storytelling** (`workflow`, `project`)：剧本与分镜管理。

* **Character Consistency** (`character_consistency`)：核心功能，确保角色在不同镜头中的一致性。

* **Video Generation** (`video_rendering`, `lip_sync`)：视频合成与口型匹配。

* **Asset Management** (`asset_library`, `sound_effect`)：素材与音效库。
