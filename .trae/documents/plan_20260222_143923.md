## 🚀 部署上线计划

### 方案选择：**云端托管方案（推荐）**

这种方式成本最低、部署最快，适合个人开发者和小团队。

***

### 第一阶段：准备工作

#### 1. 注册云服务账号

* **Supabase**（免费）：数据库 + 文件存储

* **Railway**（免费额度）：后端API托管

* **Cloudflare Pages**（免费）：前端托管

* **Upstash**（免费额度）：Redis缓存（可选）

#### 2. 获取AI服务API密钥

* **Replicate API Token**：AI图像生成

* **ElevenLabs API Key**：英文TTS语音合成

* **OpenAI API Key**：GPT功能（可选）

***

### 第二阶段：后端部署（Railway）

1. 连接GitHub仓库到Railway
2. 配置环境变量（数据库URL、API密钥等）
3. 运行数据库迁移
4. 验证API健康检查

***

### 第三阶段：数据库配置（Supabase）

1. 创建Supabase项目
2. 获取数据库连接字符串
3. 配置存储桶（用于文件上传）
4. 运行数据库迁移脚本

***

### 第四阶段：前端部署（Cloudflare Pages）

1. 连接GitHub仓库
2. 配置构建命令和输出目录
3. 设置环境变量（API地址）
4. 配置自定义域名（可选）

***

### 第五阶段：域名与HTTPS

1. 配置自定义域名
2. 设置DNS解析
3. 启用HTTPS（自动配置）

***

### 第六阶段：验证上线

1. 测试用户注册/登录
2. 测试核心功能流程
3. 检查API响应时间
4. 配置监控告警

***

### 预估成本

| 服务               | 免费额度             | 超出后费用        |
| ---------------- | ---------------- | ------------ |
| Supabase         | 500MB数据库 + 1GB存储 | $25/月起       |
| Railway          | $5免费额度/月         | 按用量计费        |
| Cloudflare Pages | 无限制              | 免费           |
| **总计**           | **基本免费**         | **约$5-30/月** |
