## 快速开始

### 第一步：注册账户

1. 访问 https://yourdomain.com
2. 点击"注册"按钮
3. 填写邮箱和密码
4. 验证邮箱
5. 完成注册

### 第二步：选择订阅计划

我们提供四种订阅计划：

| 计划 | 价格 | 额度 | 适用场景 |
|------|------|------|----------|
| 免费版 | ¥0/月 | 100分钟/月 | 个人试用 |
| 按量付费 | ¥1/分钟 | 按需购买 | 偶尔使用 |
| 专业版 | ¥299/月 | 2000分钟/月 | 专业创作者 |
| 企业版 | 定制 | 无限制 | 团队协作 |

### 第三步：创建第一个项目

1. 登录后，点击"新建项目"
2. 输入项目名称和描述
3. 选择视频比例（推荐9:16竖屏）
4. 点击"创建"

恭喜！您已经准备好开始创作了。

## 项目创建

### 创建新项目

1. 点击"新建项目"
2. 填写项目信息：
   - **项目名称**：给您的短剧起个名字
   - **项目描述**：简单描述故事内容
   - **视频比例**：选择9:16（竖屏）、16:9（横屏）或1:1（方屏）
3. 点击"创建"

### 编辑项目

1. 在项目列表中找到项目
2. 点击项目卡片进入编辑页面
3. 可以修改：
   - 项目名称和描述
   - 剧本内容
   - 角色设置
   - 分镜安排

### 删除项目

1. 在项目列表中找到项目
2. 点击"删除"按钮
3. 确认删除
4. **注意**：删除后无法恢复

### 项目协作

#### 邀请团队成员

1. 打开项目
2. 点击"协作"按钮
3. 输入成员邮箱
4. 选择权限级别：
   - **查看者**：只能查看项目
   - **编辑者**：可以编辑项目
   - **管理员**：可以管理项目和成员
5. 发送邀请

#### 管理协作者

1. 在"协作"页面查看所有成员
2. 可以修改成员权限
3. 可以移除成员

## 角色管理

### 上传角色图片

1. 进入项目编辑页面
2. 点击"角色管理"
3. 点击"上传角色"
4. 选择角色图片（支持JPG、PNG格式）
5. 输入角色名称
6. 点击"上传"

**图片要求**：
- 格式：JPG或PNG
- 大小：不超过10MB
- 分辨率：建议1024x1024或更高
- 内容：清晰的人物正面照

### 生成一致性模型

上传角色后，系统会自动生成一致性模型：

1. 提取面部特征
2. 分析服装和发型
3. 创建一致性模型
4. 处理时间约2秒

### 编辑角色

1. 在角色列表中找到角色
2. 点击"编辑"
3. 可以修改：
   - 角色名称
   - 角色描述
   - 更换参考图片
4. 保存修改

### 删除角色

1. 在角色列表中找到角色
2. 点击"删除"
3. 确认删除
4. **注意**：删除角色会影响使用该角色的分镜

## 分镜编辑

### 创建分镜

1. 进入项目编辑页面
2. 点击"分镜管理"
3. 点击"新建分镜"
4. 填写分镜信息：
   - **场景编号**：分镜的顺序
   - **场景描述**：详细描述场景内容
   - **角色选择**：选择出现的角色
   - **时长**：场景持续时间（秒）
5. 点击"创建"

### 生成分镜图像

1. 创建分镜后，点击"生成图像"
2. 选择风格：
   - **动态漫**：卡通风格
   - **真人短剧**：写实风格
3. 等待生成（约3-5秒）
4. 查看生成结果

**提示**：
- 场景描述越详细，生成效果越好
- 可以多次生成，选择最满意的结果
- 系统会自动保持角色一致性

### 编辑分镜

1. 在分镜列表中找到分镜
2. 点击"编辑"
3. 可以修改：
   - 场景描述
   - 角色选择
   - 时长
   - 重新生成图像
4. 保存修改

### 排序分镜

1. 在分镜列表中
2. 拖拽分镜卡片调整顺序
3. 系统自动保存新顺序

### 删除分镜

1. 在分镜列表中找到分镜
2. 点击"删除"
3. 确认删除

## 工作流执行

### 什么是工作流？

工作流是自动化的创作流程，包括：
1. 剧本解析
2. 角色生成
3. 分镜生成
4. 口型同步
5. 音效匹配
6. 视频合成

### 启动工作流

1. 确保项目已准备好：
   - 已编写剧本
   - 已上传角色
   - 已创建分镜
2. 点击"启动工作流"
3. 选择工作流类型：
   - **完整制作**：执行所有步骤
   - **快速预览**：只生成低质量预览
4. 点击"开始"

### 监控进度

工作流启动后：
1. 可以实时查看进度
2. 查看当前执行的步骤
3. 查看预计完成时间

### 暂停和继续

如果需要修改：
1. 点击"暂停"按钮
2. 修改项目内容
3. 点击"继续"按钮
4. 工作流从暂停处继续

### 取消工作流

如果不满意：
1. 点击"取消"按钮
2. 确认取消
3. 可以重新启动

## 视频导出

### 估算费用

导出前，系统会显示：
- 预计费用
- 剩余额度
- 是否有足够额度

### 配置导出

1. 点击"导出视频"
2. 选择导出设置：
   - **分辨率**：720p、1080p或4K
   - **格式**：MP4或MOV
   - **字幕**：是否包含字幕
3. 查看费用估算
4. 点击"确认导出"

### 等待导出

导出过程：
1. 视频渲染（约3-5分钟）
2. 实时显示进度
3. 完成后自动通知

### 下载视频

导出完成后：
1. 在"导出历史"中找到视频
2. 点击"下载"按钮
3. 视频保存到本地

### 导出历史

在"导出历史"页面：
- 查看所有导出记录
- 重新下载已导出的视频
- 查看导出详情

## 常见问题

### 账户相关

**Q: 忘记密码怎么办？**
A: 点击登录页面的"忘记密码"，输入邮箱，按照邮件指引重置密码。

**Q: 如何修改邮箱？**
A: 进入"账户设置" → "邮箱设置" → 输入新邮箱 → 验证新邮箱。

**Q: 额度用完了怎么办？**
A: 可以升级订阅计划或购买额外额度。

### 项目相关

**Q: 项目可以保存多久？**
A: 项目永久保存，除非您主动删除。

**Q: 可以恢复删除的项目吗？**
A: 删除后30天内可以联系客服恢复，超过30天无法恢复。

**Q: 项目数量有限制吗？**
A: 免费版最多5个项目，专业版和企业版无限制。

### 角色相关

**Q: 角色图片有什么要求？**
A: 建议使用清晰的正面照，分辨率1024x1024或更高，JPG或PNG格式。

**Q: 可以使用真人照片吗？**
A: 可以，但请确保您有使用该照片的权利。

**Q: 角色一致性如何保证？**
A: 系统使用AI技术提取角色特征，生成的所有分镜都会保持角色一致性（相似度>90%）。

### 分镜相关

**Q: 分镜生成需要多长时间？**
A: 通常3-5秒，取决于场景复杂度。

**Q: 生成的分镜不满意怎么办？**
A: 可以修改场景描述后重新生成，或手动上传图片。

**Q: 可以使用自己的图片吗？**
A: 可以，支持上传自定义分镜图片。

### 导出相关

**Q: 导出需要多长时间？**
A: 1-3分钟的视频通常需要3-5分钟导出。

**Q: 导出失败怎么办？**
A: 检查网络连接，确保额度充足，如果问题持续请联系客服。

**Q: 导出的视频可以商用吗？**
A: 专业版和企业版用户可以商用，免费版仅供个人使用。

### 技术相关

**Q: 支持哪些浏览器？**
A: 推荐使用Chrome、Firefox、Safari或Edge最新版本。

**Q: 支持移动端吗？**
A: 目前主要支持桌面端，移动端功能有限。

**Q: 数据安全吗？**
A: 我们使用加密存储和传输，定期备份数据，确保您的数据安全。
